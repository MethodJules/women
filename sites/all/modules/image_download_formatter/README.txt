Image Download Formatter
========================

This module adds a field formatter to display the content of an image field with a download button.

Most of the code, is just copy from the Drupal core image field formatter, so its usage and settings should be very familiar.

Usage
-----

1. Install and enable this module
2. Go to the display setting for an entity (ie. node, user) containing an image field
3. Select the "Image Download" display format for an image field
4. Optionally choose an image style for the Image Download display settings
5. Save the entity display settings
